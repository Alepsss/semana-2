Comandos utiles
COMANDO DESCRIPCIÓN
pwd muestra la ruta actual donde estas.
mkdir permite crear una carpeta.
touch permite crear un archivo.
cat permite visualizar la información de un archivo.
alias permite ver el listado de los alias creados.

Comandos alias utiles que puedes crear para utilizar en la Terminal

Para crear tu alias debe conocer primero que SHELL tienes por defecto en la terminal, para ello utilizaremos el siguiente comando:

echo $SHELL

Dependiendo de la terminal que estes usando tenemos que ejecutar algunos de estos comandos:

  • nano ~/.bashrc
  • nano ~/.zshrc
  • nano ~/.config/fish/config.fish
Alias para GNU Linux
COMANDO DESCRIPCIÓN
alias actualizar="sudo aptitude update && sudo aptitude upgrade" permite realizar la actualización de programas y paquetes instalados
alias limpiar="sudo aptitude clean && sudo aptitude autoclean" permite hacer mantenimiento a programas y paquetes instalados
